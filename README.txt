Language Used: Java
Instructions to run the code:

1) Place the java file (kmeans.java) source code in the required directory.
 
2) Compile it using the following command,  
   javac kmeans.java

3) Input arguments: <number of clusters, input data file and the output file>
   kmeans.java 20 input.txt output.txt

Here 20 is the number of clusters.
     		
(Please take the input file from our folder since our code doesn’t execute if the first line of input file contains column names)
